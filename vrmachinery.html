Tactical Simulation

VRMACHINERY is an immersive virtual reality simulation that integrates hands-on firefighting operations with real-time strategic (RTS) decision-making to provide a realistic and educational experience in fire management. Designed as both a training tool and an engaging game, FlaimeVR challenges players to manage resources, deploy firefighting units, and adapt to dynamic environmental conditions. Players experience diverse scenarios—from containing small-scale house fires to controlling fast-spreading wildfires and managing complex industrial emergencies—while facing hazards such as chemical explosions and shifting wind patterns. The inclusion of AI-driven environmental data enhances situational awareness by providing real-time updates on fire behaviour, weather conditions, and operational consequences. This combination of realistic VR immersion, strategic resource management, and AI-assisted decision support creates a unique platform for developing crisis management, leadership, and problem-solving skills.

Development Pipeline

01. 3D Modelling

Comprehensive environmental and model 3D asset creation using Blender. Focusing on making the model as accurate as possible with real life equipment.

02. Unity development

Developed within the Unity Universal Render Pipeline (URP). Features include custom HLSL shaders and a modular C# event system for XR integration.

03. AI Integration

Using Gemini as its source, the system includes an AI assistant to provide real-time guidance and feedback within the training environment.

04. Programming Logic

Clean, documented C# code structure designed for modularity and scalability using Visual Studio Code and Unity's event-driven architecture.
