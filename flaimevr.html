Tactical Simulation

FLAIMEVR is an immersive virtual reality simulation that integrates hands-on firefighting operations with real-time strategic (RTS) decision-making. Designed as both a training tool and an engaging game, FlaimeVR challenges players to manage resources, deploy firefighting units, and adapt to dynamic environmental conditions. The inclusion of AI-driven environmental data enhances situational awareness by providing real-time updates on fire behaviour, weather conditions, and operational consequences.

Development Pipeline

01. 3D Modelling

Comprehensive environmental and model 3D asset creation using Blender. Focusing on making the model as accurate as possible with real life equipment.

02. Unity development

Developed within the Unity Universal Render Pipeline (URP). Features include custom HLSL shaders and a modular C# event system for XR integration.

03. AI Integration

Using Gemini as an API, the system includes an AI assistant to provide real-time guidance and feedback within the training environment.

04. Visual Studio Code

Programming logic utilizing C#. Clean, documented code structure designed for modularity and scalability within the Unity environment.
