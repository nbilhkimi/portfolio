Tactical Simulation

FLAIMEVR is a high-fidelity firefighting simulation designed for emergency response training. Players are immersed in dynamic, physics-based fire environments where every second counts. The project focuses on realistic tool handling, thermal imaging accuracy, and procedural smoke propagation.

Developed as a professional portfolio piece, this simulation demonstrates the integration of complex VR interaction patterns with optimized visual effects for standalone hardware.

Development Pipeline

C# / XR Interaction Toolkit

01. Physics-Based Interaction

Developed custom grab mechanics for heavy equipment (axes, hoses, extinguishers) utilizing two-handed interaction logic and haptic feedback profiles to simulate weight and resistance.

Unity VFX Graph / Shader Graph

02. Dynamic Fire Systems

Created a performant fire propagation system using GPU particles and custom shaders. The heat maps dynamically react to extinguisher particles, creating a realistic "knockdown" effect.

Blender / Substance Painter

03. Industrial Asset Design

Modeled high-detail, low-poly industrial environments optimized for VR. Used PBR workflows to ensure surface textures (metal, soot, water) react accurately to the emergency lighting.
