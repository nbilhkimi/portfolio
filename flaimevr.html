Tactical Simulation

FLAIMEVR is an immersive virtual reality simulation integrating hands-on firefighting operations with real-time strategic decision-making.

Development Pipeline

01. 3D Modelling

Comprehensive environmental and model 3D asset creation using Blender. Focusing on making the model as accurate as possible with real-life equipment.

02. Unity Development

Developed within the Universal Render Pipeline (URP). Features include custom HLSL shaders for environmental effects and modular event systems for XR triggers.

03. AI Integration

Using the Gemini API, the system includes an intelligent assistant to guide users through complex training protocols in real-time.

04. Programming Logic

Logic utilizing C# via VS Code. Clean, documented code structure designed for modularity and scalability within the simulation environment.
