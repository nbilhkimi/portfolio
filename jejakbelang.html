Welcome to Jejak Belang

Embark on a heartwarming journey about Mika's adventure to find his missing cat Belang. Jejak Belang is a top down narrative-driven adventure that explores the bonds between animals and humans, combining cozy exploration with meaningful storytelling.

Players must navigate environmental puzzles, interact with eccentric characters, and find their way home in a world inspired by local heritage and soft, whimsical aesthetics.

Technical Pipeline

Adobe Illustrator & Figma

01. Visual Concepts

I crafted the cat protagonists and NPC designs in Illustrator. Figma was essential for the user interface, ensuring the "paw-sitive" experience starts from the menu.

Unity Engine

02. Technical Development

Built using Unity URP. I focused on vibrant lighting and custom shaders to bring the feline's perspective to life in a stylized 3D environment.

Blender 3D

03. Stylized 3D Creation

Environmental props and cat animations were modeled in Blender. Low-poly techniques were used to keep the performance smooth for a relaxing experience.

Visual Studio Code (VSC)

04. C# Modular Logic

Core gameplay, including feline movement and interaction systems, was scripted in C# using VS Code for a clean and efficient codebase.
