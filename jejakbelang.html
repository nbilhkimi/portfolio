Welcome to Jejak Belang

Embark on a heartwarming journey through the eyes of a stray cat navigating a beautifully stylized world. Jejak Belang is a narrative-driven adventure that explores the bonds between animals and humans, combining cozy exploration with meaningful storytelling.

Navigate environmental puzzles, interact with eccentric characters, and find your way home in a world inspired by local heritage and soft, whimsical aesthetics.

Technical Pipeline

Adobe Illustrator & Figma

01. Characters

using adobe illustrator, i draw my characters in 2d art to get a better view of the art direction. the game is a lighthearted story driven so a soft and colourful characters design is needed to gain more interest towards younger audiance while mature enough to attract older generation.

Unity Engine

02. Unity Engine Development

Developed within the Unity Universal Render Pipeline (URP). Features include custom HLSL shaders for environmental effects and a modular C# event system for gameplay triggers.

Blender 3D

03. Stylized 3D Modeling

Comprehensive environmental 3D asset creation using Blender. Focus on low-poly optimization while maintaining high visual appeal for stylized world-building.

Visual Studio Code

04. Visual Studio Code

Programming logic utilizing C#. Clean, documented code structure designed for modularity and scalability within the Unity environment.
